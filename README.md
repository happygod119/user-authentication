## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
```
https://github.com/happygod119/user-authentication.git
```
3. 安裝 npm 套件

```
npm install
```

4. 引入種子資料

```
npm run seed
```

5. 執行程式

```
npm run start
```

6.若出現此行網址代表運行順利

```
http://localhost:3000
```
